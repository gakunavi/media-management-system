## 作業手順

1. **LINE Developers Console** を開く → 対象のプロバイダー → **Messaging API のチャネル**
2. **「Messaging API設定」タブ** を開く
3. **Webhook設定** の「Webhook URL」に、以下を**そのまま**貼る

```
https://collect.asset-support.co.jp/api/ingest/line
```

> ★**`mms.asset-support.co.jp` にしないでください。**
> そちらは Cloudflare Access の保護下にあり、LINE がログイン画面（302）を受け取って
> **Webhook が全部失敗します**（実測で確認済み）。`collect.` は Access の対象外です。

4. **「更新」** を押す
5. **「検証」** ボタンを押す → **「成功」** と出れば受口まで届いています
6. **「Webhookの利用」を ON** にする（★これを忘れると、URLを入れても配信されません）

## 触らないでほしいもの

  **作り直すと MMS 側が全部 401 になります**）
