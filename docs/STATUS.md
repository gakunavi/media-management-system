docker compose ps --format '{{.Service}} {{.Image}}'
docker compose exec web sh -c "grep -rl '<入れた文字列>' /app/apps/web/.next"

## 5. ★このシステムを触る人が最初に知るべきこと

1. **未計測（記録が無い）と実測ゼロ（0件）を混同しない**（§2）。
   これを破って直客2件の成約を見逃した事故がある
2. **推測でデータを埋めない。** 判定できないものは空のまま残し、件数を画面に出す
3. **「実装した」と書いてあっても信じない。** 実データを見る、URLは実際に叩く（§4-40）
4. **進捗はチェックボックスで管理しない。実データで判定する**（§4-91）
5. **画面に開発用語を出さない**（§4-54）。「ジョブ」→「自動処理」
6. **ラボ値を体感として報告しない**（§4-106）。PSI のモバイルは低速4G相当の模擬条件

規約は §4 に 106項目ある。**すべて実際に踏んだ失敗から書かれている。**
